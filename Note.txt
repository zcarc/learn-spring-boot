4. 스프링 부트 시작하기
	1. 스프링 부트 기본 프로젝트 직접 만들어보기.
		a. Maven Project로 직접 만들기
		b. Spring Initializer로 자동으로 만들기
		
		
	
	2. 패키징으로 생성된 jar 파일로 스프링 부트 실행
		a. mvnw package *(windows) or mvn package *(mac)
		b. java -jar target\spring-boot-getting-started-1.0-SNAPSHOT.jar
		
5. 스프링 부트 프로젝트 생성기
	1. start.spring.io 에서 생성
	
6. @SpringBootApplication 애노테이션이 붙은 패키지의 클래스 파일들의 빈들만 컴포넌트로 등록된다.
